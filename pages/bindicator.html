Bindicator



Automation Using Puppeteer

Description: The main purpose of this project was to write a script using puppeteer which goes to a council website, enters the required information to obtain the bin collection dates and then sends me that information via Slack

- The script navigates to the Barnet council website and enters the required information and then stores the bin collection dates in variables which are displayed on the console.
- Those variables are then saved as a .json and sent to slack via a POST method using a webhook.
- The process of receiving messages periodically was automated by creating a cron job.
